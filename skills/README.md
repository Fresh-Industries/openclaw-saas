# Skills Directory

Agent skills for OpenClaw SaaS development - used when coding to follow best practices.

## Installed Skills

| Source | Skills |
|--------|--------|
| **agent-skills** (Vercel) | `vercel-react-best-practices`, `web-design-guidelines`, `composition-patterns` |
| **anthropic-skills** (Anthropic) | `frontend-design`, `mcp-builder`, `pdf`, `docx`, and more |
| **better-auth-skills** | `better-auth-best-practices`, `emailAndPassword`, `twoFactor`, `organization` |
| **next-skills** (Vercel) | `next-best-practices`, `next-cache-components` |
| **prisma-skills** | `prisma-database-setup`, `prisma-cli`, `prisma-client-api`, `prisma-postgres` |
| **ui-ux-pro-max** | UI/UX design patterns |

## Usage

```bash
# Read a skill when building
cat skills/agent-skills/skills/react-best-practices/SKILL.md
cat skills/anthropic-skills/skills/frontend-design/SKILL.md
cat skills/prisma-skills/prisma-database-setup/SKILL.md
cat skills/better-auth-skills/better-auth/best-practices/SKILL.md
cat skills/next-skills/skills/next-best-practices/SKILL.md
```
